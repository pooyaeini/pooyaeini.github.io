# Dr. Pooya Eini — Personal Academic Website

A fast, responsive, single-page personal website for Dr. Pooya Eini, MD —
physician-researcher in cardiac imaging and clinical artificial intelligence.

Pure static site (HTML + CSS + vanilla JS), no build step, no dependencies.

## Files
- `index.html` — page content
- `styles.css` — styling (light + dark themes)
- `script.js` — theme toggle, publication filter, scroll animations
- `assets/img/` — portraits and research figures

## Run locally
Open `index.html` in a browser, or serve the folder:
```
python -m http.server 5178
```
Then visit http://localhost:5178

## Deploy
This folder can be hosted, as-is, on any static host (GitHub Pages,
Cloudflare Pages, Netlify). No configuration required.
